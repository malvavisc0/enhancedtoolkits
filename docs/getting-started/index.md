# Getting Started

Enhanced Toolkits is a collection of **agent-ready toolkits** built on top of [`StrictToolkit`](../api/base.md).

It provides:

- **8 core toolkits** (exported from `enhancedtoolkits`)
- **9 calculator modules** under `enhancedtoolkits.calculators`

## What you can build

- Research agents (search + downloading + reasoning)
- Financial assistants (finance + calculators + reasoning)
- File-aware assistants (files + thinking)
